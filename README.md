# Hector Operações

Sistema interno para gestão de agenda, tarefas, processos e pendências recorrentes.

## Como iniciar

1. Instale as dependências.
2. Copie `.env.example` para `.env.local`.
3. Preencha as variáveis do Supabase.
4. Execute o SQL em `supabase/schema.sql`.
5. Inicie o projeto.

```bash
npm install
npm run dev
```

## Segurança

Leia `docs/SEGURANCA.md` antes de criar novas funcionalidades.

Para criar o primeiro usuário e perfil, siga `docs/SETUP_SUPABASE.md`.
